## About

This is a fully hand-coded personal portfolio website built with vanilla HTML, CSS, and JavaScript. It showcases my work as a data engineer, featuring project write-ups, architecture diagrams, and contact information — all wrapped in a dark, professional aesthetic inspired by developer tooling.

## Featured Projects

###  ETL Pipeline — Mozambican Economic Data
An ETL pipeline that processes Mozambique's economic data sourced from the World Bank. Extracts from CSV, cleans and standardises the data, loads it into a database, and answers key business questions.  
[View on GitHub](https://github.com/Thiyane24/etl-pipeline-moz-economy)

---

### Conversas de IT — ELT Pipeline
An automated ELT pipeline that pulls video metrics (views, likes, comments) from the YouTube Data API for *Conversas de IT*, a Mozambican tech podcast. Built with PostgreSQL, Apache Airflow, Docker, and a GitHub Actions CI/CD pipeline.  
[View on GitHub](https://github.com/Thiyane24/CDI-ELT-Pipeline)

---

###  Premier League API — ETL Pipeline
An end-to-end data engineering pipeline that extracts Premier League top scorer data from the API-Football API. Processes data through a medallion architecture, loads it into a local analytical warehouse, and visualises insights in Power BI.  
[View on GitHub](https://github.com/Thiyane24/ETL-pipeline-football-api)

## Running Locally

No build tools required — just open the file in your browser.

```bash
git clone https://github.com/Thiyane24/Thiyane_Xavier-Portofolio.git
cd Thiyane_Xavier-Portofolio
open src/index.html
```

Or serve it locally with:

```bash
npx serve .
```
